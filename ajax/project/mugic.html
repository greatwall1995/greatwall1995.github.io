Mugic

Mugic is a toy compiler implemented by Cheng Wan. The source language is mixed with C and Java, which is designed by Rong Ma, our teacher of compiler course, and the target code is MIPS assembly language. In this project, we design and implement a parser, an abstract syntax tree and IR language. Furthermore, we conduct a variety of optimization concerning register allocation. The work is written in Java and ANTLR.
